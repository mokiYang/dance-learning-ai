# 页面布局重构说明

## 🎨 新布局结构

按照设计稿重构了页面布局，采用了更现代化的导航方式。

### 布局层级

```
App
├── Header (顶部固定)
│   ├── Logo (左侧)
│   └── 上传按钮 (右侧)
├── NavTabs (Tab 导航)
│   ├── 首页
│   ├── 个人页
│   └── (可扩展更多)
└── MainContent (主内容区)
    └── Routes
        ├── VideoList (首页)
        ├── VideoPlayer (播放页)
        ├── VideoResult (结果页)
        └── Profile (个人页)
```

---

## 📁 新增组件

### 1. Header 组件
**位置：** `frontend/src/components/Header/`

**功能：**
- 显示 DANCEAURA Logo
- 上传视频按钮
- 固定在页面顶部

**样式特点：**
- 渐变色 Logo
- 圆角按钮
- 响应式布局

### 2. NavTabs 组件
**位置：** `frontend/src/components/NavTabs/`

**功能：**
- 首页/个人页切换
- 可扩展新的 Tab
- 高亮当前激活Tab

**样式特点：**
- 横向滚动支持
- 底部下划线高亮
- 平滑过渡动画

---

## 🔄 修改的组件

### 1. App.tsx
- 移除了底部 TabBar
- 添加了 Header 和 NavTabs
- 调整了主内容区域的布局

### 2. VideoList
- 移除了顶部标题和上传按钮（已迁移到 Header）
- 简化了组件结构
- 优化了空状态显示

### 3. App.less
- 调整了主内容区域的 margin-top
- 移除了底部 padding
- 适配新的布局结构

---

## 📱 移动端适配

### 响应式断点

- **桌面端：** > 768px
- **平板：** 480px - 768px
- **手机：** < 480px

### 适配特点

1. **Header**
   - 移动端缩小 Logo 尺寸
   - 小屏幕隐藏上传按钮文字，只显示图标

2. **NavTabs**
   - 横向滚动支持
   - 减小 Tab 间距
   - 优化触摸区域

3. **VideoList**
   - 自适应网格布局
   - 移动端减少列数
   - 优化卡片尺寸

## 🎨 设计风格

### 颜色方案

```less
// 主色调
主色: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
背景色: #f8f9fa
文字色: #333333
次要文字: #666666
边框色: #e8e8e8

// 交互状态
```

### 视觉元素

- **Logo**: 渐变色文字，加粗字体
- **按钮**: 圆角 24px，渐变背景，阴影效果
- **Tab**: 底部下划线，渐变高亮
- **卡片**: 圆角 12px，阴影效果

## 🚀 扩展 Tab 示例

如果要添加新的 Tab，修改 `NavTabs/index.tsx`:

```typescript
const tabs = [
  {
    key: '/',
    label: '首页',
    icon: '🏠',
  },
  {
    key: '/profile',
    label: '个人页',
    icon: '👤',
  },
  // 新增 Tab
  {
    key: '/explore',
    label: '发现',
    icon: '🔍',
  },
  {
    key: '/trending',
    label: '热门',
    icon: '🔥',
  },
];
```

然后在 `App.tsx` 添加对应的路由即可。
